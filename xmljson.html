XML Rules

6 important rules that must be adhered to for creating a valid XML file:
XML Rule 1: All XML elements must have a closing tag

It is illegal to omit the closing tag when you are creating XML syntax. XML elements must have a closing tag.

<body>incorrent syntax

<body>correct syntax</body>
XML Rule 2: XML tags are case sensitive

<Body>incorrent syntax</Body>

<body>correct syntax</body>
XML Rule 3: All XML elements must be properly nested

<b><i>incorrent syntax</b></i>

<b><i>correct syntax</i></b>
XML Rule 4: All XML documents must have a root element

<root>
  <child>
    <subchild>Example</subchild>
  </child>
</root>
XML Rule 5: Attribute values must always be quoted

<note date=05/05/05>
<to>John</to>
<from>Jane</from>
<body>This note has incorrect syntax</body>
</note>

<note date="05/05/05">
<to>John</to>
<from>Jane</from>
<body>This note has correct syntax</body>
</note>
