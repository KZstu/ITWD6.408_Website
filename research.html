Copyright, Creative Common (CC) Licenses, Fair Use

Basic principles of copyright, 4 types of CC licences and Fair Use

Creative Commons licenses give everyone from individual creators to large institutions a standardized way to grant the public permission to use their creative work under copyright law.
Liscenses:
CC 0: Worldwide public domain
CC BY: Credit must be given to the creator
CC BY-SA: + Adaptations must be shared under the same terms
CC BY-NC: + Only noncommercial uses of the work are permitted
CC BY-ND: + No derivatives or adaptations of the work are permitted
https://creativecommons.org/about/cclicenses/

Search Engine Optimization

5 ways to improve the visibility of your website on search engine.

  1. Place keywords in the URL, page title, and headings.
  2. Visibility, because google is a search engine, it can't see anything but text, set your navigation and links with text only
  3. Load time, google ranks websites based on how long they take for the user to load them, use smaller images and stick to a minimalistic design
  4. Create a sitemap and a Robots.txt file to correctly index your website
  5. Use an SEO checker to make sure your website doesn't have any crucial mistakes that will omit it from googles index

3 major factors you should consider when choosing a right web hosting provider

  1. Server Reliability, Uptime Scores And Security: Weak servers and unstable network connections can often push your site offline. This leads to low rankings on search engines like Google and Yahoo. Working with a hosting company with above 99.5 percent uptime is the key to your site’s success.

Web security

5 common cyber attacks (threats) and preventive measures against those attacks

SQL Injection Attackers trick the server into providing data by placing code in input fields, prevent with parameterized queries which separate SQL code from user input

DDoS Overwhelms the target’s web server with requests from a botnet, Cloudfare can help prevent this

Man in the Middle Unencrypted data can be intercepted, prevent with encryption protocols like HTTPS

Broken Authentication This can be prevented with strong passwords or 2FA

Cross Site Scripting attacks trick a browser into delivering malicious client-side scripts to the victim’s browser, this can be prevented by sanitizing user inputs
